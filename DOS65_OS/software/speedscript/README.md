SpeedScript Fork
--------------------
SpeedScript is a very small but full-featured word processor, originally written 
by Charles Brannon, and published in Compute Magazine for various 8-bit
computers. This project is a port of the Commodore 64 version of SpeedScript
to the 6502PC running DOS/65 with the memory mapped video card.

SpeedScript 3.1 for the Commodore 64 was the starting point. It is
one patch away from being SpeedScript 3.2, but this is the version which
had the source code published in print.  You can read the original book
[on archive.org]: https://archive.org/details/Computes_Speedscript



The program works in 80 column mode. 

Command Keys:

* `<RIGHT>`:cursor right
* `<LEFT>`: cursor left
* `<DEL>`:  delete character at cursor
* `Ctrl+D`: delete (S,W,P)
* `Ctrl+S`: home (top of text)
* `<INS>`:  insert mode toggle
* `Ctrl+Q`: erase all text (with confirm)
* `Ctrl+E`: erase (S,W,P)
* `Ctrl+L`: load file
* `Ctrl+V`: save file
* `Ctrl+R`: insert buffer
* `Ctrl+X`: switch (cut to buffer)
* `Ctrl+Z`: go to end of text
* `Ctrl+\`: format code
* `Ctrl+Y`: delete line
* `Ctrl+A`: alpha (go to top of screen)
* `Ctrl+K`: kill buffer
* `Ctrl+H`: hunt/search (backspace handled before here)
* `Ctrl+_`: show free memory
* `Ctrl+C`: tab
* `Ctrl+J`: replace start
* `Ctrl+G`: search and replace
* `<DOWN>`: paragraph down
* `<UP>`:   paragraph up
* `Ctrl+W`: word left
* `Ctrl+U`: word right
* `<F3>`:   return to os
* `<F1>`:   disk catalog
* `<F9>`:   print
